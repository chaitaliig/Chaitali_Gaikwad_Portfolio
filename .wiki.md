# Project Summary
The project is a portfolio website for **Chaitali Gaikwad**, a Data Analyst & ML Engineer. It showcases her professional experience, skills, projects, and education, aimed at attracting potential employers and educational institutions in Germany. The site features a modern design with a dark navy theme, responsive layout, and user-friendly navigation.

# Project Module Description
The portfolio consists of several functional modules:
- **Hero**: Introduction section with name, title, tagline, and call-to-action buttons.
- **About**: Professional summary and quick facts.
- **Experience Timeline**: Details of previous roles with highlights.
- **Projects & Case Studies**: Displays project cards with detailed modals.
- **Skills & Tools**: Categorized skill sets with proficiency badges.
- **Education, Certifications & Publications**: Academic background and certifications.
- **Contact**: Contact form and links to social profiles.
- **Footer**: Navigation links and compliance notices.

# Usage
1. Install dependencies:
   ```bash
   cd app/frontend
   pnpm install
   ```
2. Build the application:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run start
   ```
